Advanced Topics Journal - Week of February 19 to 25, 2017

Day 1: The Plan

I have two goals for the coming week:

  • Learn to the latest CSS layout techniques using two tutorials - A Complete Guide to Flexbox and A Complete Guide to Grid. Both of these tutorials are from the wonderful https://css-tricks.com website.
  • Continue to make progress going through The Flask Mega-Tutorial. I am currently in the middle of Part IV: Database and have been posting my progress to a github repository here.

I wanted to use both grid layout and media queries to style the pages for the Unsung Sheroes of Computer Science project for our computer lab. After some quick investigation, I came across this page, which makes clear that I'm a wee bit early on using grid. It will be supported in Firefox 52. I'm currently running Firefox 51, so support is on its way, but I perfer to wait at least until the current Ubuntu version of Firefox support what I'm learning.

Instead of a using a grid layout, I just applied the table-row and table-cell display types to child elements of a figure element to achieve the layout I wanted. I started with Edzon's work on the Unsung Sheroes and modified it to produce this result.

Day 3

During class I varified that students could login to the local server and see there web files. Tomorrow I'll show students how to use the rsync utility to synchronize files between the two servers.
