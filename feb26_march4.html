12:30 pm - A Shared Screen Session

Six of us shared a single screen session and wrote Python together to transform an augmented matrix into reduced row echelon form. After modifying .screenrc and .bash_aliases as described in yesterday's post, here is the process:

Screen Sharer Setup

Use ssh to connect from the presentation laptop to the in class web server and run the following commands: 

$ sudo chmod 755 /var/run/screen
$ screens sharedscreen

Connecting to the Shared Screen

Users can then connect to the shared screen by sshing into the classroom webserver and running: 

$ sreen -x [username of screensharer]/

Note: the username/ following -x assumes only a single screen is being shared. If there are more than one shared screens, then: 

$ sreen -x [username of screensharer]/[screenname]

would be required.
