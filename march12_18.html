Snow day today, so I'll have time to do some studying! ;-)

ER Diagrams

First order of business is to identify the best Ubuntu tool for creating ER models. A quick search led me to Umbrello, which I installed with: 

$ sudo apt install umbrello

The ER diagram documentation has a screenshot that appears promising.

The task now will be to get Martin to learn to use this! I came up with a project for Owen and Martin to work on which I am calling Virginia Votes. The goal will be to setup a database using publically accessible data sets to provide voters with information about their state and local elections and elected officials.

Postgis Server

For a project like VA Votes, we will need a publically available database, preferably PostgreSQL with geospatial extensions (i.e. PostGIS). I woke up this morning intending to setup one up on a server I already have available, and found out I had done this on the intended server already. Next time we have class I'll get Martin setup as a database admin on it.
